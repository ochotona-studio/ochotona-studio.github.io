# aura:pureplay

**순수하게 음악 재생에 집중하는 MP3 플레이어**

aura:pureplay는 이름 그대로 '순수한 재생(Pure Play)'에 집중한 오디오 플레이어 앱입니다.
복잡한 기능 없이 음악 파일 재생에 필요한 핵심 기능만을 담았습니다.

---

## ✨ 주요 기능

### 🎵 폴더 기반 음악 재생
- 기기에 저장된 음악 파일을 폴더별로 탐색
- 최근 재생한 곡 빠른 접근

### 🌓 라이트 / 다크 모드 지원
- 사용 환경에 맞게 테마 전환 가능
- 눈의 피로를 줄이는 다크 모드 제공

### ⏱️ 슬립 타이머
- 설정한 시간 후 자동 재생 중지
- 취침 전 음악 감상에 최적화
